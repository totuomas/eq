## A Parametric 3-Band Audio Equalizer Browser Extension

A lightweight, cross-browser audio equalizer for Chromium and Firefox using the Web Audio API

## Getting Started

### Installation

1. Clone the repo

* `git clone https://github.com/totuomas/eq.git` **OR** [Download ZIP](https://github.com/totuomas/eq/archive/refs/heads/main.zip) and unzip folder.

2. Install project locally

* Go to `chrome://extensions/` in Chrome (or `about:addons` in Firefox).

* Enable **Developer mode** (top-right in Chrome).

* Click **Load unpacked** and select the project folder.

### Tips

Works best with YouTube audio.

## ✨ Features

* 🎚️ **Customizable frequency bands**

  * Adjust gain and Q factor for precise audio control  
* 🔊 **Preamp and master volume**

  * Fine-tune overall audio output  
* 📊 **Real-time frequency response graph**

  * Visualize your adjustments instantly  
* 🌈 **Live spectrum analyzer**

  * Bar-style visualization of audio spectrum  
* 🗂️ **Per-tab state management**

  * Keeps settings unique per browser tab  
* 🎛️ **Preconfigured presets**

  * Quickly switch between sound profiles
